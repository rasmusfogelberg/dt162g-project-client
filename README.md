# DT162G PROJECT ASSIGNMENT
## Client

This repo contains the client part. It is built with React and TypeScript. A user can without a login create a workout and then enter exercises he or she have done for the session. When the user saves his or hers workout it is saved and can be view and edited after.

The API repo for this assignment can be found here:
[https://github.com/rasmusfogelberg/dt162g-project-api.git](https://github.com/rasmusfogelberg/dt162g-project-api.git)
